Программное обеспечение для анализа данных видеоокулографии ETRAN

На данном ресурсе представлено описание основных возможностей бесплатно распространяемого и расширяемого программного обеспечения ETRAN с открытым исходным кодом, предназначенного для анализа данных видеоокулографии.

Разрабатываемое ПО является попыткой воплощения системы для анализа данных видеоокулографии, соответствующей основным требованиям, определённым авторами при разработке концепции таких систем (Мармалюк и др., 2015; Zhegallo & Marmalyuk, 2015): кросплатформенность, наличие базовых статистических методов и критериев, расширяемость и открытость программного кода. Исходный программный код системы и пользовательское руководство доступны по следующим ссылкам:

  1. Как начать пользоваться ETRAN

  2. Репозиторий с исходным программным кодом разработанной системы ETRAN

На данном ресурсе перечислены основные возможности ПО ETRAN, которые позволяют решать следующие задачи:

  1. Загрузка данных и стимулов

  2. Редактор областей интереса

  3. Предобработка данных

  4. Детекция событий

  5. Выборка данных и событий

  6. Расчёт показателей

  7. Расчёт сложных представлений

  8. Визуализация

  9. Разведочный анализ

В разделе "Примеры использования (use-cases)" приводятся примеры результатов обработки данных.

Ресурс разработан при поддержке Российского гуманитарного научного фонда (проект №14-06-12012 «Программное обеспечение с открытым исходным кодом для анализа результатов окулографических исследований»).

Ресурс содержит информацию о программном обеспечении ETRAN, разработанном в ходе указанного проекта. ETRAN распространяется бесплатно, как есть, без каких-либо ограничений распространения и модификации. Доработки программного обеспечения ETRAN не предусмотрены, однако, любой желающий может создать копию репозитория проекта и вносить любые изменения в программный код ETRAN и документацию к нему.
